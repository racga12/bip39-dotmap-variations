⚠️ Critical Security Advice Before Introducing Words
1. Run Completely Offline

Disconnect your computer from the internet (air-gap) or run on a secure offline Live OS like Tails or Ubuntu Live.

2. Disable Browser Extensions

Open this app in a Private / Incognito Window. Third-party extensions can scrape page inputs and leak secrets.

3. Type, Do Not Paste

Type your recovery words manually instead of copy-pasting to prevent clipboard histories or background tools from caching your phrase.

Please follow these advices and use it at your own risk.

🔒 Critical Security Actions After Generating Output
1. Clear Clipboard

Copy some random dummy text (e.g., "abc") to completely wipe your clipboard history buffer.

2. Close Windows

Completely close all browser tabs and clear the temporary browser memory cache.

3. Kill Server

Terminate the local Flask backend server process running on your machine immediately.

4. Restart Device

Reboot/restart your machine to guarantee that system RAM is completely flushed and cleared.
